# 聊天记录持久化功能指南

## 功能概述

UniAudioAgent 现已支持多个聊天记录的持久化保存。所有对话会自动保存到本地存储，重启应用后会自动恢复。

## UI 布局

```
┌─────────────────────────────────────┐
│ UniAudioAgent │  [≡] [⚙️]            │
├──────────────┬──────────────────────┤
│  + 新建对话   │  当前对话内容         │
│              │  ─────────────────   │
│  对话 1 ✎ ✕   │  消息列表             │
│  对话 2 ✎ ✕   │                      │
│  对话 3 ✎ ✕   │  [输入框]           │
│              │                      │
└──────────────┴──────────────────────┘
```

## 主要功能

### 1. 创建新对话
- 点击左侧边栏的 **"+ 新建对话"** 按钮
- 系统自动创建新对话并切换至该对话

### 2. 切换对话
- 点击左侧边栏列表中的任意对话项
- 对话内容立即切换，当前对话会高亮显示（青色背景）

### 3. 重命名对话
- 将鼠标悬停在对话项上，点击 **编辑图标 (✎)**
- 输入新的对话标题
- 按 Enter 确认或 Esc 取消

### 4. 删除对话
- 将鼠标悬停在对话项上，点击 **删除图标 (✕)**
- 确认删除后，系统会自动切换到其他对话
- 如果删除了最后一个对话，系统会自动创建新对话

### 5. 清空对话
- 在聊天窗口右上角点击 **"清空对话"** 按钮
- 仅清空当前对话的消息，对话本身不被删除

### 6. 隐藏/显示侧边栏
- 点击标题栏左边的 **菜单图标 (≡)** 来切换侧边栏显示/隐藏
- 这样可以获得更大的聊天窗口空间

## 数据持久化

- 所有聊天记录自动保存到 `electron-store` 数据库
- 数据使用操作系统级别的加密存储
- 应用关闭后重启，所有对话会自动恢复

## 技术细节

### 新增 IPC 通道
- `CHAT_SESSIONS_LIST` - 获取所有对话列表
- `CHAT_SESSIONS_CREATE` - 创建新对话
- `CHAT_SESSIONS_LOAD` - 加载特定对话
- `CHAT_SESSIONS_DELETE` - 删除对话
- `CHAT_SESSIONS_SAVE` - 保存对话

### 存储结构
```
{
  chatSessions: [
    {
      id: "uuid",
      title: "对话标题",
      messages: [...],
      createdAt: timestamp,
      updatedAt: timestamp
    }
  ],
  currentSessionId: "uuid"
}
```

## 快捷键（计划中）

- `Ctrl+N` - 新建对话
- `Ctrl+L` - 清空对话
- `Ctrl+Shift+D` - 删除对话

## 已知限制

1. 对话列表按创建/更新时间排序
2. 单个对话的消息数量没有限制
3. 暂不支持对话导出/导入

## 故障排除

**Q: 对话未保存？**
A: 检查确保在消息发送完成后（显示"对话完成"）再关闭应用。

**Q: 找不到之前的对话？**
A: 检查侧边栏是否被隐藏，点击菜单按钮显示侧边栏。

**Q: 意外删除了对话？**
A: 当前版本无撤销功能，建议定期备份数据文件。
